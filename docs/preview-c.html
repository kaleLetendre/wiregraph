The pipeline

Indexed once. Always fresh.

Your source walks through tree-sitter into a local SQLite graph, exposed to Claude as MCP tools. Edit code and it re-indexes.

The differentiator

Links code across repos — through contracts.

A contract is the seam between two compartments — the communication that is not a code-level call. When a distinctive token shows up in 2+ repos, wiregraph snaps them into a link.

Two compartments · zero direct calls · one shared token → one link.

Get started

Four lines. Then forget it.

Install the plugin, reload, and initialize. wiregraph builds the graph and keeps it fresh on its own.

claude code
›/plugin marketplace add kaleLetendre/wiregraph
›/plugin install wiregraph@wiregraph
›/reload-plugins
›/wiregraph-init

What happens

  1. marketplace add — points Claude at the wiregraph plugin.
  2. install + reload — loads the MCP tools.
  3. wiregraph-init — walks your repos, builds graph.db, wires auto-update.
The graph is a SQLite file in your workspace. Nothing is uploaded.
